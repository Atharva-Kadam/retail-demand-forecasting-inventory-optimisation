# Retail Demand Forecasting & Inventory Optimisation

## Overview
End-to-end data analytics project focused on improving retail operations through demand forecasting and inventory optimisation.

## Objective
Analyse historical sales data to:
- Reduce stockouts
- Optimise inventory levels
- Improve operational efficiency

## Tools Used
- Python (Pandas)
- SQL
- Tableau

## Key Insights
- Sales peak in December (seasonality)
- Promotions increase sales by ~100%
- Demand varies across weekdays
- Top stores drive majority of revenue

## Inventory Optimisation
- Safety stock calculated using demand variability
- Reorder points defined
- EOQ principles applied
